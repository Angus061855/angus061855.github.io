# 雙職缺廣告落地頁設計

## 目標

建立一個獨立、可投放廣告的長篇落地頁，同時介紹酒店公關與紓壓按摩美容師。主要轉換是點擊個人 LINE 詢問，不要求使用者先填表。

## 核心訊息

- 先詢問再決定
- 不用和經紀人或店家簽約
- 不壓薪水
- 兩種工作方向，由本人一對一說明後再選
- 工作範圍、安全與隱私在面試前說清楚

## 頁面結構

1. 首屏：品牌、信任數字、雙主標、四項承諾、LINE CTA
2. 常見擔心：合約、喝酒、隱私、薪資與拒絕權
3. 雙職缺比較：酒店公關與美容師並列，說明工作型態、排班與薪資方式
4. 選擇引導：依個人時間、是否喝酒及工作接受度判斷方向
5. 關於大頭：七年經驗、超過 3000 位個人面試、本人回覆
6. 與一般經紀的差異：不催促、不隱瞞扣款、不用話術綁人
7. 適合與不適合：降低無效詢問並建立信任
8. FAQ：第一次詢問、試做、合約、離職、隱私、薪資
9. 最終 CTA：個人 LINE、官方 LINE、八大影片、官方 IG

## 視覺

- 沿用 AS娛樂黑底、金色、絲絨光影與現有字體
- 不複製參考網站的酒紅色與人物照片
- 使用現有黑色絲絨影像作為首屏實際素材
- 卡片半徑不超過 6px，重點警示保留低飽和紅色
- 桌面版保持長篇閱讀節奏；手機版以單欄、雙欄比較與固定底部 LINE CTA 提升轉換

## 技術

- 新增獨立靜態頁 `work-landing.html`
- CSS 與 JavaScript 寫在頁面專屬檔案，避免影響主網站
- 使用既有 `site-config.js` 統一官方 IG
- 所有外部連結使用 `noopener noreferrer`
- 支援 `prefers-reduced-motion`
- 保留 Meta Pixel PageView，LINE 按鈕加入可辨識的轉換事件名稱

## 驗證

- Playwright 驗證 1440×1000、430×932、375×812
- 檢查無水平捲動、按鈕可點、文字不破版
- 檢查 LINE、官方 LINE、TikTok、Instagram 連結
- 檢查首屏在手機版能看到主標與主要 CTA
