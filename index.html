入行知識

酒店生存大百科

入行前必懂的術語與眉角
點擊展開查看　刮開學姐防雷建議

一節 / 節薪 / 打卡 / 報班 / 門禁

● 一節　台北的酒店　都是以 10 分鐘為一個單位

● 節薪　每一節給的薪水　臺北便服店或禮服店的薪水通常落在 230 到 290 元之間

● 打卡　上班時會給你一張卡片　放入打卡機成功打卡後　再交給控台

● 報班　帶台會在妳上班時詢問下禮拜有哪幾天可以上班　只要報確定能上班的那幾天就好了

● 門禁　告訴控台　妳今天要在幾點前下班回家

一檔 / 壓檔

● 一檔　酒店的專用上班期間單位　每一個禮拜叫做一檔　每一檔的薪水　會在下一檔的時候拿到　一個月會有四檔

● 壓檔　公司沒有給妳上一檔的薪水　這叫做壓檔

點擊刮開：防雷建議
壓檔就是在侵犯公關的權益 在AS娛樂裡 絕不會發生
薪水看天數

指薪水按上班天數調漲　例如一檔上三天班節薪是 230　四天班 240　五天班 250

點擊刮開：防雷建議
強烈建議先問清楚節薪待遇 並且留存聊天紀錄
薪水直走

不管上幾天班　妳的薪水都是固定的

點擊刮開：經驗談
節薪並不會因為是直走的就變得很少　直走現領的薪水一樣可以很高
現領

當天上班　當天下班就領薪水　或是隔天匯款　不會拖太久

點擊刮開：現領隱藏陷阱
要搞清楚經紀是提供永久現領　還是僅限試上期間現領
另外有些經紀會說現領需要降節薪　有些則提供現領不降節薪　一開始就先問清楚
上卡 / 下卡 / 休檔

● 上卡　來這家店上班　面試通過　取好名字並填好資料

● 下卡　就是在這家店離職的意思

● 休檔　休息一陣子不上班　但沒有離職

制服店

主打性感　尺度較大　提供秀舞服務以及手工

點擊刮開：小介紹
條件無法達到禮服店水準　但極度想上班　且能接受這種尺度的人才會去　制服店節薪會比禮服店低
禮服店

商務會館　穿著正式小禮服　強調氣質談吐　服務包含聊天　喝酒　唱歌　互動尺度中等

點擊刮開：新手村推薦
建議新人先去禮服店磨練一個月　等有工作經驗了
如果長相條件確實可以　再去框店面試　比較不建議毫無經驗就直接衝框店
小框店 / 便服店

對素質要求很高　不由客人排排站挑選　而是由幹部帶入包廂
因為客人的消費能力高以及店家的機制　使得被框的機率會提高　所以叫做小框店

大框店 / 便服店

酒店業界頂端　進場客人經濟實力極強　核心制度在於全場買斷　客人一旦中意通常會直接框到底

點擊刮開：業界冷知識
有些框店又被稱作砲店　指接任務的公關居多　不接任務且長相普普的話可能上不好
但如果長得美若天仙　就算不接任務也都可以上得好
八大按摩 / 紓壓會館

提供全身油壓按摩　重點在後續加值服務　需上空但不用喝酒
這幾年因為不用喝酒吸引較多人加入　整體素質慢慢提升　條件太差也不一定能面試進了

行政 / 帶台

● 行政　負責排妹　帶妹妹進入包廂看台的人

● 帶台　專門負責妳的行政　處理報班　上下班　累了醉了直接跟他說

幹部 / 控台

● 幹部　帶客人來消費的人　對幹部有禮貌　他們才會推薦優質客人給妳

● 控台　負責打卡與調度　下台後去找控台　若廣播聽到空台看台　代表有新包廂需要選公關了　要立刻前往

少爺 / 公主

負責掃地　換毛巾冰塊　點餐　換麥克風電池等雜事

點擊刮開：做人處事提醒
他們工作非常辛苦　就算喝了酒　也務必要記得對他們態度好一點　記得跟他們說謝謝
明配 / 暗配

● 明配　不管是什麼客人基本上都會接任務的公關

● 暗配　會看跟客人相處下來的感覺　再決定要不要接任務的公關

上台 / 空台 / 看台 / 順位

● 上台　被客人選中坐在包廂裡

● 空台　沒被選中　空台時沒薪水

● 看台　進包廂給客人挑的過程

● 順位　看包廂的順序　絕對不可以跳號

下台 / 咖掉 / 自卡

● 下台　客人離開後去找控台報下桌

● 咖掉　頻率不合請妳離開　離開後務必回報行政或控台

● 自卡　客人太色或喝太多　委婉撒嬌仍無法應付時　去廁所傳訊息給行政求救帶妳離開

灣掰

包廂只有一位客人　且客人只點了一位小姐

點擊刮開：自保求生守則
可以選擇上班前先講清楚不坐灣掰桌　若坐下才發現　可按鈴請少爺進來換東西
或經同意叫行政排妹　人員進出能增加保障　減少獨處時間
預三節 / 要續嗎 / 不續

坐滿 3 節約 30 分鐘時　行政會進來問客人要續嗎

● 不預　滿意　不用再問了

● 再預三　再觀察 30 分鐘

點擊刮開：留客小技巧
不續就是請妳離開　千萬別在客人還沒回答時就拿包包作勢要走　可以適當撒嬌求留　成功的話親親他臉頰說謝謝哥哥
滿包 / 塞包 / 佔包

生意好沒空包廂時　客人會請行政排妹妹先進去坐著佔包

點擊刮開：黃金十分鐘
當客人進來時要趕快站起身迎接討歡心　因為 10 分鐘後行政就會進來問要不要留下來　這時間非常重要
遞衛生紙

客人去上廁所時　趕快拿三張衛生紙折好　站在廁所門口等他出來遞給他

千萬不要呆坐在沙發上　不然客人的朋友很容易覺得妳在發呆

順坐 / 前桌 / 後桌

● 順坐　看台後被客人選中　順利坐在包廂裡

● 前桌　妳現在正在待著坐檯的這個包廂

● 後桌　其他想要點妳　找妳過去的包廂客人

點台 / 加點

● 點台　後桌指名要找妳

● 加點　前桌客人決定留下妳

點擊刮開：高情商撒嬌術
後桌點台時　請妳勾緊前桌客人的手臂　淚眼汪汪看向他
撒嬌說妳想留下來　不管點台或加點客人都有額外付錢　記得敬酒說謝謝
小框 / 大框 / 到底 / 內全

● 小框　客人買斷妳一段時間　通常 2 到 3 小時　不能再去其他包廂

● 大框　客人買斷整個晚上　直到打烊為止

● 到底　跟大框意思相同　框到打烊

● 內全　框店專用　客人包下全場所有公關

出場 / 任務 / 私接

● 出場 / 任務　下班後跟客人離開場所　提供店外服務　完全自願　任何人不得強迫

● 私接　私下跟客人聯繫接案　繞過店家

點擊刮開：重要提醒
私接被店家發現會直接下卡　且失去店家保護　出場前務必告知信任的人你的位置
喝酒技巧 / 假喝 / 換酒

● 假喝　嘴唇碰杯但不真的喝下去　或只抿一小口

● 換酒　趁客人不注意把自己的酒換成水或果汁

點擊刮開：保護自己的方法
以上我都不推薦 唯一建議就是直接跟客人說你喝不下了
客人最多把你換掉而已 他很喜歡你也會把你留下來
如果他發現你假喝、偷倒酒 到時候發現了也尷尬
小費 / 打賞 / 包紅包

客人額外給的現金　不經過店家　全數歸妳

點擊刮開：收小費眉角
收到小費要立刻說謝謝並敬酒　不要當場數錢　直接放進包包　對給小費的客人要特別記住臉　下次見到主動打招呼
養客 / 熟客 / 指名

● 養客　維持與客人的關係　讓他下次還想點妳

● 熟客　固定會來找妳的客人

● 指名　客人進場直接點名要妳

點擊刮開：養客心法
記住客人喜歡喝什麼酒　聊什麼話題　下次見面主動提起　讓他覺得妳有在乎他　熟客是最穩定的收入來源
🚨 這些合約絕對不能簽 🚨

遇到以下任何一條　立刻離開　不要猶豫　這是保護妳自己最重要的事


  • 要求簽賣身契或任何形式的人身約束合約
  • 要求繳交保證金　押金　服裝費才能上班
  • 要求提供身分證正本或讓對方保管證件
  • 合約中有違約金條款　且金額高得離譜
  • 要求簽署出場同意書或任務相關文件
  • 對方不讓妳帶走合約副本或拒絕讓妳看清楚內容
  • 要求妳交出手機或限制人身自由

合法的經紀不需要靠合約控制妳　真正好的經紀　靠的是信任與誠實　不是白紙黑字的束縛
